Hexo+GitHub部署个人blog

安装hexo

  1. 前置：安装好node，npm等。
  2. 切换到root用户： sudo su
  3. 安装镜像：npm install -g cnpm –registry=https://registry.npm.taobao.org
  4. 安装hexo框架：cnpm install -g hexo-cli

使用hexo

  1. 建立文件夹: mkdir blog
  2. 初始化hexo: sudo hexo init
  3. 开启服务hexo: hexo server (hexo s)

新建一篇文章:

  1. hexo n “我的第一篇博客文章”
  2. 更改内容: vim
  3. 清除一下缓存: hexo clean
  4. 生成一下: hexo g
  5. 本地查看一下: hexo s

部署到远端github：

  1. 创建一个仓库：必须是：你的昵称/github.io
  2. 安装部署插件：cnpm install –save hexo-deployer-git
  3. 设置_config.yml中的Deployment中的
    type: git
    repo: 仓库地址
    branch: master
  4. 部署: hexo deploy (hexo d)
  5. 可以查看 你的昵称/github.io网址，则可以看到你的博客。

自定义主题：

  1. github.com/litten/hexo-theme-yilia
  2. 下载主题：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  3. 修改_config.yml的theme: yilia
  4. 清除一下: hexo clean
  5. 生成一下: hexo g
  6. 查看一下: hexo s
  7. 推到远端: hexo d (推送远端后，需要2min左右才能看到变化)
